Инструкция по загрузке сайта на Netlify:

1. Создай новый публичный репозиторий на GitHub.
2. Загрузи в него два файла:
   - index.html
   - document.pdf
   - README.txt
3. В Netlify выбери "Add new site" → "Import from Git".
4. Выбери свой репозиторий.
5. Build command: (оставить пустым)
6. Publish directory: .
7. Нажми "Deploy site".
8. Готово — получишь ссылку на сайт.
